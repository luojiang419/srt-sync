- 字幕余量（前后 500ms）、首条字幕偏移校准

#### 2. `lib/providers/timeline_provider.dart`
- `TimelineState`: timelineList / isBuilding / isTrimming / isExporting / trimProgress
- `buildTimeline()`: 构建时间线
- `batchTrim()`: 批量裁切（实时进度更新）
- `exportXml()`: Premiere XML 导出
- 统计: totalCount / totalDurationMs

#### 3. `lib/widgets/timeline_preview.dart`
- Canvas 自绘时间线可视化
- 左侧文件名列表 + 右侧时间轴条
- 视频条（深蓝色）+ 音频条（高亮红色）+ 偏移虚线（蓝色）
- 图例说明
- `_TimelinePainter`: CustomPainter 实现毫秒级精度的缩放绘制

#### 4. `lib/widgets/step_timeline.dart`
- 顶部工具栏：重新构建 / 裁切导出 / 导出 XML
- 统计信息条（对齐数 / 总时长 / 导出路径）
- 裁切进度条（实时进度 + 当前文件名）
- 底部确认完成按钮 → 更新工程状态为 completed
- 自动构建时间线（initState）

#### 5. `lib/services/export_service.dart`
- **exportFcxml()**: 生成 FCPXML 1.9 格式（Premiere Pro / DaVinci Resolve 可导入）
- 包含 format / media / sequence / spine / lane 完整结构
- 视频片段按时间顺序排列在 spine 主轨道
- 音频片段按 offsetMs 对齐在各 lane
- **exportSrt()**: SRT 字幕文件导出
- 毫秒→帧数转换（24fps）→ FCPXML 时间码格式
