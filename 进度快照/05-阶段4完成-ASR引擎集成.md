## 本次完成内容：阶段4 - ASR引擎集成

### 新增文件 (4个)

- `lib/services/sherpa_onnx_service.dart` - sherpa-onnx 引擎服务
  - 路径检测（findExecutable / findModelDir / findVadModel）
  - 环境验证（checkEnv → SherpaOnnxEnv）
  - 命令行调用（recognize）支持 VAD 参数
  - 输出解析支持3种格式：JSON / [start,end] text / seg_id start end text

- `lib/services/asr_batch_service.dart` - 批量ASR识别服务
  - 批量编排（batchRecognize）支持进度回调 + 取消检查
  - 单文件流程：提取WAV → ASR识别 → 保存到数据库
  - 缓存检查（已识别文件自动跳过）
  - 失败重试（reRecognizeFile）
  - 临时WAV文件自动清理
  - AsrBatchResult 汇总结果

- `lib/providers/asr_process_provider.dart` - ASR识别过程状态管理
  - AsrProcessState：isRunning / overallProgress / fileProgresses
  - startBatchRecognize：启动批量识别
  - cancelRecognize：取消识别
  - reRecognizeFile：单文件重试
  - 自动读取 settings（sherpaOnnxPath / vadMode）
  - 自动更新工程状态（recognizing → recognized）

- `lib/widgets/asr_progress_panel.dart` - 单文件进度面板组件
  - 状态图标 + 状态标签（7种状态）
  - 进度条（提取/识别/保存阶段显示）
  - 错误信息显示 + 重试按钮
  - 完成后显示段落数

- `lib/widgets/step_recognize.dart` - ASR识别步骤页面
  - 标题栏 + 环境状态指示器（就绪/异常/未配置）
  - 开始识别 / 取消 / 重新识别 按钮
  - 识别完成后显示"进入下一步"按钮
  - 总体进度条 + 完成数/失败数统计
  - 文件列表（每个文件一个 AsrProgressPanel）

### 修改文件 (2个)

- `lib/screens/project_screen.dart` - Step 2 占位替换为 StepRecognize 组件
- `lib/providers/project_detail_provider.dart` - 新增 confirmRecognize() 方法
