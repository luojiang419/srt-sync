# 进度快照 40 - Dock 栏调整为软件底部显示

## 版本信息
- 版本号: v1.0.15
- 备份: backup/v0.23.0
- 编译: dist/v1.0.15

## 已完成内容

### 1. Dock 样式已从左侧栏改为底部显示
- 之前的 Dock 样式是：
  - 左侧窄栏图标导航
- 现在已改为：
  - 软件底部 Dock 导航
  - 模块切换入口统一放到底部操作区
- 这样更符合 Dock 的使用习惯，也避免和左侧信息栏抢位置

### 2. Dock 模式下左侧栏已改回信息侧栏
- Dock 模式启用后，左侧不再显示竖向图标 Dock
- 左侧现在保留为：
  - 工程摘要
  - Dock 模式提示卡片
  - 工作区概览
  - 左下角样式切换按钮
- 当前模块的切换改由底部 Dock 完成

### 3. 底部操作区已支持“Dock + 下一步按钮”并存
- Dock 模式下，软件底部现在同时包含：
  - 左侧 Dock 图标导航
  - 右侧主动作按钮（进入下一步 / 完成工程）
- 标准菜单样式下，底部仍保持原有的主动作按钮布局，不受影响

### 4. Dock 模式提示文案已补齐
- 在 Dock 模式下，左侧会明确提示：
  - 当前已启用底部 Dock 导航
  - 模块切换入口位于软件底部
- 这样切换到 Dock 样式后，不会误以为左侧功能菜单丢失

### 5. 版本与发布产物已更新
- `pubspec.yaml` 已更新为 `1.0.15+15`
- 设置页关于信息已更新为 `v1.0.15`
- 已完成 Windows release 构建并复制到：
  - `dist/v1.0.15`
