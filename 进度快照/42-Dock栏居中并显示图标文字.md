## 已完成内容

### 1. 底部 Dock 已调整为居中显示
- 之前底部 Dock 虽然位于下方，但整体偏左
- 现在已改为：
  - 宽窗口下 Dock 组居中显示
  - 右侧主动作按钮独立停靠
- 这样视觉重心更稳定，也更符合 Dock 的使用习惯

### 2. 每个 Dock 入口已改为“图标 + 文字”
- 之前 Dock 入口是纯图标按钮
- 现在每个模块入口都会直接显示：
  - 图标
  - 模块名称文字
- 这样用户无需悬停或记忆图标语义，就能直接看懂当前入口含义

### 3. 底部 Dock 按钮尺寸已扩大，便于点击
- 为了承载文字并提升可用性
- 当前 Dock 按钮已从小图标块改为更宽的卡片式按钮
- 同时保留：
  - 选中态高亮
  - 已完成模块状态点

### 4. 窄窗口下已补自适应布局
- 宽窗口下：
  - 居中 Dock + 右侧主动作按钮并存
- 窄窗口下：
  - Dock 居中显示
  - 主动作按钮自动换到下方/右侧安全位置
- 避免文字版 Dock 挤压后与主按钮重叠

### 5. 版本与发布产物已更新
- `pubspec.yaml` 已更新为 `1.0.17+17`
- 设置页关于信息已更新为 `v1.0.17`
- 已完成 Windows release 构建并复制到：
  - `dist/v1.0.17`
