# 进度快照 65 - 复核全屏页与XML命名优化版本发布完成

## 版本信息
- 新版本号: `v1.1.9`
- `pubspec.yaml`: `1.1.9+27`
- 阶段备份: `backup/v0.35.0`
- 发布目录: `dist/v1.1.9`

## 已完成内容

### 1. 合板详情与复核已改为全屏页
- 原先的复核弹窗已改为整页 `Navigator.push(...)` 打开
- 现在进入的是全屏复核页，不再受弹窗尺寸限制
- 返回后会回到原工程页，并保留原工程上下文

### 2. 全屏复核页已承接原有复核能力
- 保留了现有的：
  - 默认第一组合板锚点
  - 稳定锚点跳转
  - 全部锚点预览列表
  - 搜索与锚点兼容
  - 键盘 `Esc / ← / →`
  - 复核动作后不自动退出
- 页面头部已增加返回按钮，满足全屏使用场景

### 3. XML 导出名称已支持默认工程名和自定义
- 导出 XML 时，会先弹出名称输入框
- 默认名称改为当前工程名称
- 用户可以自定义输入导出名称
- 确认名称后，再选择导出目录
- 最终仍会生成同名基底的一对文件：
  - `.xml`
  - `.fcpxml`

### 4. XML 名称清洗规则已接入
- Windows 非法文件名字符会自动替换为 `_`
- 会去掉首尾空格并压缩多余空白
- 清洗后为空时，会回退到工程名称
- 工程名称也不可用时，会回退到 `ASR Timeline`

### 5. 构建与启动冒烟已完成
- `flutter build windows --release`

- 已生成：
  - `build/windows/x64/runner/Release/asr_tools.exe`

- 已完成启动冒烟：
  - `dist/v1.1.9/asr_tools.exe`
