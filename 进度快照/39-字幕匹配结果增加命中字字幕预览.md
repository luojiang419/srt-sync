# 进度快照 39 - 字幕匹配结果增加命中字字幕预览

## 版本信息
- 版本号: v1.0.14
- 备份: backup/v0.22.0
- 编译: dist/v1.0.14

## 已完成内容

### 1. 字幕匹配结果现在会直接显示“命中的字幕内容”
- 之前点击 `开始匹配` 后，只能看到：
  - 哪个视频文件匹配到了哪个音频文件
  - 置信度高低
- 现在每条匹配结果卡片下方会直接显示：
  - 视频命中字幕
  - 音频命中字幕
  - 匹配相似度
  - 偏移时间

### 2. 不改数据库结构，直接按现有字幕数据生成预览
- 这次没有新增数据库字段
- 改为在 UI 渲染匹配结果时：
  - 重新读取当前视频/音频的字幕片段
  - 计算最佳命中字字幕窗口
  - 截取 1~3 条最像的字幕片段做直观预览
- 这样能最快把“到底匹配了哪些字幕”展示出来

### 3. 匹配结果卡片已升级为“文件 + 字幕预览”双层展示
- 顶部仍然保留：
  - 视频文件名
  - 音频文件名
  - 置信度
  - 确认按钮
  - 字幕对比按钮
  - 删除按钮
- 下方新增：
  - `命中字幕预览` 区块
  - 左右并排的字幕片段内容
  - 时间范围标签
  - 相似度与偏移信息

### 4. 预览加载异常也有明确提示
- 如果某条匹配结果无法生成字幕预览
- 卡片内会直接显示加载失败或无可视化字幕证据
- 不会一直空白或只转圈

- `pubspec.yaml` 已更新为 `1.0.14+14`
- 设置页关于信息已更新为 `v1.0.14`
