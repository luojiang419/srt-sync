# 进度快照 43 - 长期素材字幕锚点合板主流程改造

## 版本信息
- 当前基础版本: v1.0.17
- 阶段备份: `backup/v0.26.0`

## 已完成内容

### 1. 数据层升级
- `media_files` 已扩展：
  - `sort_index`
  - `layout_start_ms`
  - `layout_end_ms`
  - `frame_rate`
  - `sample_rate`
  - `channels`
  - `width`
  - `height`
  - `has_embedded_audio`
  - `file_size`
  - `modified_at_ms`
- 新增模型与表：
  - `subtitle_files`
  - `source_layouts`
  - `subtitle_windows`
  - `match_candidates`
  - `sync_results`
  - `anchor_pairs`
  - `timeline_items`
- 已补数据库迁移逻辑，旧库可升级到新结构

### 2. 字幕准备链路已接通
- 新增 `SubtitlePrepareService`
- 已实现：
  - SRT 解析
  - 素材平铺表生成
  - 总字幕 / 单素材字幕识别
  - 总字幕反解回单条素材字幕
  - 跨边界字幕切分与归属
  - 音频字幕窗口索引生成（1/3/5/7 句）

### 3. 一键合板引擎已替换
- 旧 `MatchPair` 主逻辑已被 `SyncResult` 主结果链路取代
- 已实现：
  - 单条视频独立检索音频字幕库
  - 候选评分
  - 锚点求解
  - `audio_source_in/out` 计算
  - 越界修正
  - 高/中/低/无匹配状态输出
  - 手动改配入口

### 4. 主界面四步流程已切到新语义
- `StepImport` 已改为四分区：
  - 视频素材
  - 音频素材
  - 视频字幕
  - 音频字幕
- 已支持：
  - 文件/目录导入
  - 顺序上移下移
  - manifest 覆盖排序
  - 字幕类型手动改判
- `StepRecognize` 已改为“字幕准备”
- `StepMatch` 已改为“一键合板 + 异常与手动改配”
- `StepTimeline` 已改为“时间线与导出”

### 5. 导出链路已升级
- 时间线已改为围绕 `SyncResult` 构建
- 已支持导出：
  - 精简版 XML
  - 审片版 XML
  - CSV 报告
  - 分素材 SRT
- 已把 marker 文案接入导出数据
