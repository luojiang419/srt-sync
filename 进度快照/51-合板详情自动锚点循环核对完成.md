# 进度快照 51 - 合板详情自动锚点循环核对完成

## 已完成内容

### 1. 合板详情与复核弹窗已增加“合板锚点”按钮
- 顶部已增加 `合板锚点` 按钮
- 当当前结果存在可解析锚点时：
  - 首次显示 `合板锚点 1/N`
  - 点击后会按顺序循环所有可核对锚点
  - 继续点击会跳到下一条锚点

### 2. 自动合板锚点已支持映射到右侧整轨总字幕
- 若锚点本身就是总字幕 id，则直接使用
- 若锚点指向当前已匹配音频的局部字幕，则会自动映射回整轨总字幕
- 映射失败的锚点会自动过滤，不参与循环跳转

### 3. 点击锚点按钮后已同步联动左右字幕与预览区
- 左侧视频字幕会跳到对应锚点
- 右侧音频总字幕会跳到对应锚点
- 当前锚点字幕会自动设为选中态
- 预览区会同步刷新到该锚点对应的结果
- 搜索框内容和命中高亮会保留，不会被清空

### 4. 已补锚点排序规则
- 主锚点优先级按以下顺序确定：
  - 与 `audioSourceInMs` 距离最近
  - `similarity` 更高
  - `videoTimeMs` 更早
- 第一次点击默认先跳主锚点
