## 已完成内容

### Flutter 项目创建
- `flutter create --platforms=windows` 创建 Windows 桌面项目
- Flutter 3.38.8 / Dart 3.10.7
- 配置 pubspec.yaml（Riverpod, GoRouter, sqflite_common_ffi, file_picker, window_manager, google_fonts, uuid, process_run, crypto, xml 等依赖）
- `flutter pub get` 安装全部依赖
- `flutter build windows --debug` 编译成功

### 窗口配置
- windows/runner/main.cpp: 窗口标题 "ASR合板工具"，大小 1280x800，居中显示

### Core 层 (4个文件)
- `lib/core/constants.dart` - 路径常量、匹配阈值、VAD参数预设（标准/长音频两个模式）、VadPreset 类
- `lib/core/app_theme.dart` - 深色专业工具主题（DaVinci Resolve 风格），Material3
- `lib/core/app_router.dart` - GoRouter 路由表：/ → /project/:id → /settings
- `lib/core/extensions.dart` - String/Int/Double 扩展方法

### 入口文件 (2个文件)
- `lib/main.dart` - 初始化 window_manager + DatabaseService，启动 AsrToolsApp
- `lib/app.dart` - ProviderScope + MaterialApp.router + 暗色主题

### 数据模型 (5个文件)
- `lib/models/asr_project.dart` - AsrProject + ProjectStatus 枚举 (7个状态)
- `lib/models/media_file.dart` - MediaFile + MediaType/SubtitleStatus 枚举
- `lib/models/subtitle_clip.dart` - SubtitleClip + SRT 格式转换
- `lib/models/match_pair.dart` - MatchPair + 置信度等级
- `lib/models/timeline_data.dart` - TimelineData + FFmpeg 裁切参数生成

### 服务层 (1个文件)
- `lib/services/database_service.dart` - SQLite 完整 CRUD：
  - 4张表建表 + 索引
  - Project: 增删改查
  - MediaFile: 按项目/类型查询
  - SubtitleClip: 批量插入
  - MatchPair: 按置信度排序查询

### 占位页面 (3个文件)
- `lib/screens/home_screen.dart` - 首页占位（暗色主题 + 新建按钮）
- `lib/screens/project_screen.dart` - 工程页占位
- `lib/screens/settings_screen.dart` - 设置页占位

### 备份
- `backup/v0.1.0/` - 当前版本源码已备份
