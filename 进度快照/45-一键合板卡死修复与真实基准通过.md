# 进度快照 45 - 一键合板卡死修复与真实基准通过

## 版本信息
- 当前开发版本: `v1.1.1`
- 阶段备份: `backup/v0.27.0`

## 已完成内容

### 1. 一键合板已改为后台 worker 执行
- `SubtitleMatchService.matchProject()` 已接入 isolate worker
- UI 主线程不再直接承担候选比较与锚点计算
- 已支持：
  - 阶段进度回传
  - 当前视频名回传
  - 取消合板

### 2. 候选比较已增加预筛和限流
- 已增加廉价预筛：
  - 仅比较同 `windowSize`
  - 文本长度差比例过滤
  - 无 bigram 交集过滤
  - `Dice` 预分阈值过滤
- 已增加命中限流：
  - 每个 `videoWindow` 对单个 `audioFileId` 最多保留 8 条
  - 每个 `audioFileId` 总命中上限 48 条

### 3. 锚点搜索已改为粗偏移附近局部搜索
- `_buildAnchors()` 不再默认全音频全文扫
- 现在优先围绕 `fallbackOffsetMs` 在局部半径内搜索
- 局部无结果时才回退到全列表

### 4. 高频短句已做降权
- 已对低价值短句加入权重抑制：
  - `嗯`
  - `啊`
  - `哦`
  - `好`
  - `行`
  - `操`
  - `我操`
  - `可以`
  - `开始`
  - `等一下`
- 仅影响窗口 `uniquenessWeight`
- 不直接删除字幕文本
