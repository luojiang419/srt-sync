## 已完成内容

### 1. 合板详情与复核已改为全屏页
- 原先的 `showDialog(...)` 复核弹窗已改为整页 `Navigator.push(...)` 打开
- 现在进入的是全屏复核页，不再受弹窗尺寸限制
- 返回后会回到原工程页，并保留原筛选和模块状态

### 2. 全屏复核页已承接现有复核能力
- 保留当前已有的：
  - 默认第一组合板锚点
  - 稳定锚点滚动定位
  - 预览区全部锚点列表
  - 键盘 `Esc / ← / →`
  - 搜索与锚点兼容
  - 复核动作后不自动退出
- 页面头部已加入返回按钮，满足整页使用场景

### 3. XML 导出名称已改为工程名称默认值
- 导出 XML 时会先弹出名称输入框
- 默认值改为当前工程名称，不再使用工程 ID
- 用户可手动自定义导出名称
- 确认名称后，再选择导出目录

### 4. XML 导出名称清洗规则已补充
- 会将 Windows 非法文件名字符 `\\ / : * ? \" < > |` 替换为 `_`
- 会去掉首尾空格并压缩多余空白
- 若清洗后为空，则回退到工程名称
- 若工程名称也不可用，则回退到 `ASR Timeline`

### 5. 自动化验证已完成

  - 通过
- `flutter test --no-pub`
