## 已完成内容

### 1. 一键合板页重复标题已移除
- 已移除内容区顶部重复显示的“大号一键合板”标题
- 保留说明文字和右侧 `一键合板 / 重新合板` 操作按钮

### 2. 合板详情与复核弹窗已改为持久化显示
- 点击遮罩或弹窗外部区域不再自动关闭
- 仅允许通过右上角关闭按钮或按下 `Esc` 关闭弹窗
- `接受 / 移除 / 恢复 / 匹配` 成功后不再自动关闭弹窗

### 3. 合板详情与复核弹窗已支持连续核对
- 头部新增 `上一条` / `下一条` 按钮
- 支持键盘 `← / →` 按当前筛选列表顺序切换不同素材
- 到达列表边界时不循环切换，直接禁用对应方向
- 搜索框获得焦点时，左右键不拦截，仍交给输入框使用

### 4. 复核动作后的自动跳转已完成
- 在 `待复核` 等筛选列表中，当前条目处理后如果不再属于当前列表：
  - 优先跳到下一条
  - 若没有下一条，则跳到上一条
  - 若当前筛选列表已空，则保留弹窗并停留在当前条目，禁用上下条切换
